API参数配置
名称 默认值 类型 描述
opacity .6 number 全局透明度
color [] 即随机色 array 设置颜色数组，粒子则按设置的颜色数组随机
speed 1 number 粒子运动速度
num .12 number 粒子个数，默认为容器的0.12倍，传入[0, 1)显示容器相应倍数的值，或传入具体个数[1, +∞)
max 2.4 number 粒子最大半径
min .6 number 粒子最小半径
dis 130 number 两个点之间相连接的最大距离
lineWidth .2 number 连接线段的宽度
r 160 number 范围越大，连接的点越多
eventElem null null 或 element 触发移动事件的元素，null为canvas自身，或传入原生元素对象，如document
resize true boolean 自适应窗口尺寸变化
