API参数配置
名称 默认值 类型 描述
opacity 1 number 全局透明度
color [] array 或 string
  • 线条的颜色，空数组则随机取色，或传入依次定义线段的颜色的数组，数组里的颜色比线条个数少，系统即随机生成颜色对应赋值。
  • 传入一个颜色字符串，则所有线条应用此颜色。（以下有些雷同参数如lineWidth等，同理，不再赘述）
num null null 或 number 线条的个数，null表示随机个
lineWidth [] array 或 number 线条的宽度，空数组则随机(.2,2)的宽度，或依次定义线段的宽度
offsetLeft [] array 或 number 线条的横向偏移值，(0, 1)表示偏移波长的倍数，0 & [1, +∞)表示具体数值
offsetTop [] array 或 number 线条中点到元素顶部的距离，(0, 1)表示容器高度的倍数，0 & [1, +∞)表示具体数值
crest [] array 或 number 波峰数值，(0, 1)表示容器高度的倍数，0 & [1, +∞)表示具体数值
rippleNum [] array 或 number 波纹个数，即正弦周期个数
speed [] array 或 number 运动速度
area false boolean 是否绘制成区域图，类似水切面的效果
stroke true boolean 是否绘制边框，绘制边框即我们看到的线条
resize true boolean 自适应窗口尺寸变化
