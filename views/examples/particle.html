API参数配置
名称 默认值 类型 描述
num .12 number 粒子个数，默认为容器宽度的0.12倍个。
传入(0, 1)显示容器宽度相应倍数的个数，
传入[1, +∞)显示具体个数
max 2.4 number 粒子最大半径
min .6 number 粒子最小半径
dis 130 number 粒子两圆心点连线的最大值， 在range范围内的两圆心点距离小于dis，则两粒子之间连线
lineWidth .2 number 线段的宽度
range 160 number 定位点内连线的范围，以半径计算，范围越大连线越多，当range等于0时，不连线，相关值无效。
(定位点：即鼠标移动到画布时的鼠标点，一开始默认生成一点)
speed 1 number 粒子运动速度
eventElem null null 或 element 改变定位点坐标的事件元素，null表示canvas画布，或传入原生元素对象，如document等
API公共参数 （为什么会出现这个？这个是什么？）
如无特别说明，像opacity，color，resize这三个API在每个pjs插件配置上都存在， 为了节省重复劳动，已将参数说明信息放置在 【5分钟上手Particleground.js -> 配置 -> 2】下， 但是担心第一次点击实例及API导航进来不知是咋回事， 故再多出现一次，往后的实例将不再出现。
名称 默认值 类型 描述
opacity 1 number 画布全局透明度，取值范围：[0-1]
color [] string 或 array
粒子颜色
  • 空数组表示随机取色
  • 特定颜色的数组，如：['red', 'blue', 'green']，则表示粒子颜色为这三种色值的随机一种
  • 当为string类型时，如：'red'，则表示粒子都填充为红色
resize true boolean 自适应窗口尺寸变化
