开启/Open
暂停/Pause
源代码：
                        
    var d = new Particleground.particle( '#demo', {
        dis: 90,
        r: 100,
        resize: true
    });
    //暂停
    d.pause();
    //开启
    d.open();
API配置参数，如未特别说明，参数均为可选，包括配置对象。
名称 默认值 类型 描述
opacity .8 number 全局透明度
color [] 即随机色 array 设置颜色数组，粒子则按设置的颜色数组随机
speed 1 number 粒子运动速度
num .1 number 粒子个数，默认为容器的0.1倍，传入[0, 1)显示容器相应倍数的值，或传入具体个数[1, +∞)
max 2.4 number 粒子最大半径
min .6 number 粒子最小半径
dis 130 number 两个点之间相连接的最大距离
lineWidth .2 number 连接线段的宽度
r 240 number 范围越大，连接的点越多
eventElem false boolean 或 element 触发移动事件的元素，false为canvas，或传入原生元素对象，如document
resize false boolean 自适应窗口尺寸变化
方法，如无特别说明，所有粒子运动对象都带有暂停与开启运动的方法，后面的例子将省略此表格，避免重复劳动。
名称 描述
.open() 开启粒子运动
.pause() 暂停粒子运动
