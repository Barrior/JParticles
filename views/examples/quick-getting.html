1、获取Particleground.js（以下简称pjs）
通过以下两种方式获取pjs：

1、从官网下载界面选择特定的特效打包下载，或者直接下载包含所有特效的文件。

2、克隆pjs在Github上的整个项目，在public/js/pjs文件夹下选择你需要使用的js文件。

2、引入pjs
在页面内使用script标签引入插件js即可，当然也支持模块化的加载方式，不过pjs仍然在window对象上注册了Particleground对象，用于插件拓展。

            


            

                

                    3、创建特效
                

                

                    准备一个具有一定宽高的DOM元素，用于显示粒子特效。

                    使用 new Particleground.特效名 创建特效。
                

                

            


            

                
这样我们的粒子运动特效就完成了！

                                .pause()
                                暂停粒子运动
                            

                            

                                .open()

                    配置
                

                

                    new Particleground.pluginName( selector [, options] );
                

                

                    
                        

                            参数
                            类型
                            描述
                        

                    
                    
                        

                            selector
                            string 或 element
                            必填，选择器字符串或DOM元素
                        

                        

                            options
                            object
                            可选，特效配置参数

                    Particleground.particle.configDefault = {
                        opacity: .6,
                        color: [],
                        speed: 1,
                        num: .12,
                        max: 2.4,
                        min: .6,
                        dis: 130,
                        lineWidth: .2,
                        r: 160,
                        eventElem: null,

                    };
